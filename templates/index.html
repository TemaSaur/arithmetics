Используя шаблоны Jinja2 можно подставить нужные данные. Например:

Cтраница была загружена {{ counter }} раз с запуска сервера. Для отображения этого числа можно написать {% raw %}{{ counter }}{% endraw %}


Также можно писать условия

{% raw %}{% if number > 5 %} {{ number }} {% else %} не больше пяти {% endif %}{% endraw %}

number =

Результат:

{% if number > 5 %} {{ number }} {% else %} не больше пяти {% endif %}


Также можно писать циклы похожие на код Python

{% raw %}{% for x in range(10) %} {{ x }} {% endfor %}{% endraw %}

Результат:

{% for x in range(10) %} {{ x }} {% endfor %}


И конечно же можно итерировать по спискам

{% raw %}{% for x in collection %} {{ x }}<br> {% endif %}{% endraw %}

Результат:

{% for x in collection %} {{ x }}
{% endfor %}
